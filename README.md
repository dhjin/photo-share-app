# PhotoShare App

React Native (Expo) 앱 — 위치 기반 힐링 사진 P2P 공유 플랫폼.

## 주요 기능

- **인증** — 이메일/비밀번호 로그인·회원가입
- **피드** — 주변 사진 / 확산 랭킹 전환
- **지도** — 사진 위치를 지도에서 탐색, 노드 수 표시
- **업로드** — 촬영 또는 갤러리에서 선택, 자동 위치 태깅, 압축 처리
- **P2P 다운로드** — WebRTC를 통해 캐시 노드에서 직접 다운로드
- **캐시 노드 참여** — 다운로드한 사진의 캐시 노드가 되어 포인트 적립
- **사진 요청** — 특정 위치 사진 요청 → 근처 사용자 알림 → 포인트 보상
- **프로필** — 포인트 현황, 리워드 시스템 설명

## 실행 방법

```bash
cp .env.example .env
# EXPO_PUBLIC_API_URL을 서버 주소로 수정

npm install
npm start
```

## 화면 구성

| 탭 | 설명 |
|---|---|
| 🏠 피드 | 주변/랭킹 사진 목록 |
| 🗺️ 지도 | 위치 기반 사진 지도 |
| 📷 업로드 | 사진 촬영·업로드 |
| 📝 요청 | 사진 요청 생성·응답 |
| 👤 프로필 | 포인트, 설정 |

## WebRTC P2P 흐름

```
다운로더 → 시그널링 서버 → 캐시 노드
              ↓
        offer/answer 교환
              ↓
        ICE candidates 교환
              ↓
        DataChannel 직접 연결
              ↓
        사진 바이트 전송
              ↓
        다운로더 → 새로운 캐시 노드로 등록
```
